🐍 Python Report Generator

The report.py Python script connects to the same MySQL database and generates a full clinic report. Run it from your terminal after setting up the database.

python/report.py — Usage # Install the required library pip install mysql-connector-python # Print today's report to the terminal python report.py # Report for a specific date python report.py --date 2025-01-15 # Save the report to a .txt file python report.py --save # Both: specific date and save to file python report.py --date 2025-01-15 --save

📁 Project Structure clinic/
├── index.html           ← Frontend (this file)
├── php/
│   ├── config.php     ← DB connection (PDO)
│   └── api.php       ← REST API (patients/doctors/appointments)
├── python/
│   └── report.py     ← Report generator (mysql-connector-python)
└── sql/
    └── clinic_db.sql ← Database schema + sample data
⚙️ Setup Instructions
  1. Import sql/clinic_db.sql into MySQL (via phpMyAdmin or CLI)
  2. Update credentials in php/config.php and python/report.py
  3. Place index.html + php/ folder in your web server root (XAMPP/WAMP/Apache)
  4. Update the API_BASE constant in this HTML file to point to your server
  5. Run pip install mysql-connector-python for the Python script
